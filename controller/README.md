# Camada Controller (Controle)

Esta pasta contém as classes responsáveis por intermediar as requisições do usuário.

- **Responsabilidade:** Receber os dados da `View`, processar a lógica de decisão e chamar os métodos necessários da `Model`.
- **O que colocar aqui:** Arquivos como `UsuarioController.php`, `ProdutoController.php`, etc.
- **Regra:** Não deve conter comandos SQL nem código HTML pesado.
