# Camada Model (Modelo)

Esta pasta é o coração da lógica de dados e regras de negócio da aplicação.

- **Estrutura interna:**
    - `dao/`: Classes de acesso a dados (Data Access Object).
    - `dto/`: Objetos de transferência de dados (Data Transfer Object).
- **Responsabilidade:** Gerenciar como os dados são validados e como eles interagem com o banco de dados.
