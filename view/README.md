# Camada View (Visão)

Esta pasta contém a interface do usuário.

- **Responsabilidade:** Exibir os dados ao usuário final e enviar as interações para o `Controller`.
- **O que colocar aqui:** Arquivos PHP/HTML que compõem as telas do sistema.
- **Dica:** Tente manter o mínimo de lógica PHP aqui, apenas o necessário para loops de exibição e ecoar variáveis.
