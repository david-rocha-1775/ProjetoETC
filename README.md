# Projeto ETC - Padrão MVC
projeto de conclusao do curso tecnico

Este projeto utiliza a arquitetura **Model-View-Controller (MVC)** para separar as preocupações e facilitar a manutenção.



[Image of MVC architecture diagram]


### Estrutura Principal:
- **index.php**: Ponto de entrada da aplicação (Front Controller).
- **control/**: Lógica de controle.
- **model/**: Camada de dados e conexão.
- **view/**: Interface do usuário.
- **assets/**: Arquivos estáticos (CSS, JS, Imagens).
