# Templates de Layout

Arquivos de estrutura HTML compartilhados por todas as Views.

- **header.php**: Abre o HTML, inclui o `<head>`, o `<header>` com navegação e o sistema de mensagens.
- **footer.php**: Fecha o `<main>`, adiciona o `<footer>` e fecha o HTML.
- **Como usar:** Toda View deve incluir `header.php` no início e `footer.php` no final.
